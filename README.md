# W2M Thailand Contract Rates 2027

Static GitHub Pages site for:

- `Excursion & Package`
- `Transfer Service`
- `All Programs` download package 

## Publish With GitHub Pages

1. Create a GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub: `Settings > Pages`
4. Set:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` (or your default branch)
   - `Folder`: `/ (root)`
5. Save and wait for the Pages URL.

## Custom Domain Later

When `w2mthailand.com` is active:

1. Open `Settings > Pages`
2. Add custom domain: `w2mthailand.com`
3. Then update DNS at Z.com to point to GitHub Pages
4. Enable HTTPS after DNS finishes propagating
